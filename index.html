Подход

Ключевые решения и рабочий подход.

Технический подход в Cinema Notes

Expo + React Native клиент, Laravel API и дополнительный API-слой для отдельных сценариев и публикация APK через GitHub Releases.

Технический подход в EDO Portal

Laravel 10, Livewire 2, SQL-база, маршруты документов, отделы, роли, фильтрация статусов и интерактивные таблицы на серверном рендеринге.

Технический подход в Meteor Mod

Minecraft Forge 1.21, Gradle, Java event subscribers, жизненный цикл кастомной entity, сетевой пакет для flash-эффекта и изменения мира после удара.

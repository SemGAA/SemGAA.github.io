Реализация

Ключевые решения в проектах.

Архитектура Cinema Notes

Expo + React Native client, Laravel API, Cloudflare Worker as edge/runtime experiment, release notes and APK distribution through GitHub Releases.

Архитектура EDO Portal

Laravel 10, Livewire 2, SQL database, document routes, departments, roles, status filtering and server-rendered interactive tables.

Minecraft Forge 1.21, Gradle, Java event subscribers, custom entity lifecycle. network packet for flash feedback and world impact effects.
